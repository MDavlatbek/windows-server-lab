Windows Server Lab (Home Lab)
Описание проекта

Этот проект представляет собой домашнюю лабораторию, в которой я изучаю основы администрирования Windows Server в виртуальной среде (VirtualBox).

Цель проекта

Практическое изучение базовых задач системного администратора:

развертывание Windows Server
настройка Active Directory
управление пользователями и группами
настройка сетевых сервисов (DNS, DHCP)
работа с правами доступа
Используемые технологии
Windows Server (версия)
VirtualBox
Windows 10/11 (клиентские машины)
Active Directory Domain Services
DNS
DHCP
Что было сделано
Развернут виртуальный сервер Windows Server
Создан домен (Active Directory)
Добавлены пользователи и группы
Настроены политики паролей
Настроен общий доступ к папкам
Настроены базовые сетевые параметры (IP, DNS)
Результаты

Получен практический опыт работы с базовой инфраструктурой Windows Server и понимание принципов работы доменной сети.

Примечание

Проект учебный и создан для практики системного администрирования.
